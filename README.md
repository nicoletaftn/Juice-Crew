# Juice-Crew

A small static “meet the team” page for [OWASP Juice Shop](https://owasp.org/www-project-juice-shop/) — a fictional crew behind the intentionally vulnerable juice store. Used together with OWASP Juice Shop web application to create a more realistic scenario for the evaluation of the AI pentester together with the scope enfocements implemented.

## What's inside

- `**people-page/**` — HTML, CSS, and vanilla JS
- Six fictional team members with bios, tags, and “Learn more” modals

No dependencies.
